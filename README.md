# 🔍 Job Seeker Web Application

## 📌 Project Overview
The Job Seeker Web Application is designed to help users explore and manage job opportunities easily.  
The application provides a simple interface where users can add, view, update, and delete job listings using a database-connected system.

---

## 🎯 Objective
- Provide a simple platform to manage job opportunities
- Demonstrate Python database connectivity with MySQL
- Help beginners understand CRUD operations in web applications
- Build a practical project using Streamlit

---

## ⚙️ Key Features
- 💼 Add new job listings
- 📄 View available jobs
- ✏️ Update job information
- ❌ Delete job postings
- 🗄️ MySQL database integration
- 🌐 Simple web interface using Streamlit

---

## 🛠️ Technologies Used
- Python
- Streamlit
- MySQL
- Python Database Connectivity (PDBC)
- VS Code
- Git & GitHub

---

## 🧠 Working Principle
1. The user opens the Streamlit web application.
2. The application connects to the MySQL database.
3. Users can perform operations such as:
   - Add new job entries
   - View job listings
   - Update job information
   - Delete job records
4. The database updates instantly based on user actions.
